la télévision des tressaillements - Twitch.tv

'Use the Twitchtv JSON API'

Quatrième exercice dans la série des 'Intermediate Development Projects' du programme freecodecamp.org.

Après une introduction succincte aux fonctionnalités asynchrones de JavaScript (AJAX), il s'agit de composer une page présentant une liste de fournisseurs diffusant des informations sur une chaîne de télévision en ligne, 'Twitchtv'.

Des informations relatives à cette activité sont fournies par une API.

'user stories' de l'exercice :

  1. Je peux constater si Free Code Camp diffuse des informations
  2. Je peux cliquer sur l'indication d'activité et être directement dirigé vers la chaîne en question
  3. Si un utilisateur de 'Twitch' diffuse des informations, je peux voir s'afficher des détails sur le programme.
